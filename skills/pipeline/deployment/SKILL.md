# Deployment Skill

The Deployment agent takes the tuned, evaluated, and explained model and makes it
available as a live service. Instead of running a script manually each time you
want a prediction, the deployed model sits ready to receive new data and return
predictions instantly — from any application, tool, or system that can make a
web request.

The key principle here is: **the deployed model must behave identically to the
trained model.** This means the same scaler, the same feature order, the same
preprocessing — applied consistently to every piece of new data.

---

## Responsibilities

1. Explain what deployment means to the user in plain English
2. Load the tuned model and saved scaler from the session directory
3. Generate a FastAPI application with prediction and health check endpoints
4. Apply input validation — reject malformed requests with clear error messages
5. Apply the saved scaler to all incoming data before prediction
6. Generate a Dockerfile for containerised deployment
7. Start the API locally and run a health check
8. Test the API with a sample prediction
9. Provide the user with clear instructions on how to use their API
10. Document the API inputs and outputs in plain English

---

## What Deployment Means — Plain English

Always explain this to the user before starting:

"Until now, the model only exists as a file on your computer. Deployment means
turning it into a live service — like a web address you can send data to and
receive predictions back from.

Once deployed, you or anyone you choose can send new customer data to the API
and receive a prediction instantly, without needing to run any code themselves.
The model will be available at a web address on your machine. We can also package
it in a container so it can be moved to a server or cloud provider later.

Before we deploy, we will run a test to confirm the API is working correctly."

## Generating the FastAPI Application

```python
def generate_api_code(model_path, scaler_path, feature_columns,
                       target_col, task_type, session_id):
    """
    Generate a complete FastAPI application as a Python file.
    Written to sessions/{session_id}/api/app.py
    """

    has_scaler = scaler_path is not None
    is_classifier = "classification" in task_type

    code = f'''"""
Auto-generated prediction API
Model: {model_path}
Generated by the Data Science Pipeline
"""

from fastapi import FastAPI, HTTPException
from pydantic import BaseModel, validator
from typing import Optional, Dict, Any
import pandas as pd
import pickle
import logging
from pathlib import Path

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

app = FastAPI(
    title="Prediction API",
    description="Accepts data and returns model predictions.",
    version="1.0.0"
)

# Load model and scaler at startup
MODEL_PATH  = Path("{model_path}")
{"SCALER_PATH = Path('" + scaler_path + "')" if has_scaler else "SCALER_PATH = None"}

FEATURE_COLUMNS = {feature_columns}
TARGET_COL      = "{target_col}"

try:
    with open(MODEL_PATH, "rb") as f:
        model = pickle.load(f)
    logger.info("Model loaded successfully")
except Exception as e:
    logger.error(f"Failed to load model: {{e}}")
    raise

{"try:" if has_scaler else "scaler = None"}
{"    with open(SCALER_PATH, 'rb') as f:" if has_scaler else ""}
{"        scaler = pickle.load(f)" if has_scaler else ""}
{"    logger.info('Scaler loaded successfully')" if has_scaler else ""}
{"except Exception as e:" if has_scaler else ""}
{"    logger.error(f'Failed to load scaler: {e}')" if has_scaler else ""}
{"    raise" if has_scaler else ""}


class PredictionRequest(BaseModel):
    """Input data for a single prediction."""
    data: Dict[str, Any]

    @validator("data")
    def validate_features(cls, v):
        missing = [c for c in FEATURE_COLUMNS if c not in v]
        if missing:
            raise ValueError(
                f"Missing required fields: {{missing}}. "
                f"Required fields are: {{FEATURE_COLUMNS}}"
            )
        return v


class PredictionResponse(BaseModel):
    """Prediction result."""
    prediction: Any
    {"probability: Optional[float] = None" if is_classifier else ""}
    message: str


@app.get("/health")
def health_check():
    """Check that the API is running and the model is loaded."""
    return {{
        "status":  "healthy",
        "model":   str(MODEL_PATH.name),
        "message": "The prediction API is running and ready to accept requests."
    }}


@app.post("/predict", response_model=PredictionResponse)
def predict(request: PredictionRequest):
    """
    Make a prediction for a single row of data.

    Send a JSON object with a 'data' field containing your feature values.
    Example:
    {{
        "data": {{
            "feature_1": value,
            "feature_2": value,
            ...
        }}
    }}
    """
    try:
        # Build DataFrame in correct column order
        X = pd.DataFrame([request.data])[FEATURE_COLUMNS]

        # Apply scaling if a scaler was saved
        if scaler is not None:
            numeric_cols = X.select_dtypes("number").columns.tolist()
            X[numeric_cols] = scaler.transform(X[numeric_cols])

        # Predict
        prediction = model.predict(X)[0]

        {"# Probability for classification" if is_classifier else ""}
        {"prob = float(model.predict_proba(X)[0].max()) if hasattr(model, 'predict_proba') else None" if is_classifier else "prob = None"}

        return PredictionResponse(
            prediction={"int(prediction)" if is_classifier else "float(prediction)"},
            {"probability=prob," if is_classifier else ""}
            message=f"Prediction complete."
        )

    except ValueError as e:
        raise HTTPException(status_code=422, detail=str(e))
    except Exception as e:
        logger.error(f"Prediction error: {{e}}")
        raise HTTPException(status_code=500,
                            detail="An error occurred during prediction.")


@app.get("/features")
def list_features():
    """List the fields required to make a prediction."""
    return {{
        "required_fields": FEATURE_COLUMNS,
        "target":          TARGET_COL,
        "message": (
            f"Send these {{len(FEATURE_COLUMNS)}} fields in the 'data' object "
            f"to the /predict endpoint."
        )
    }}
'''

    return code
```

def generate_dockerfile(session_id):

## Writing and Starting the API

```python
import subprocess
import time
import requests
from pathlib import Path

def write_api_files(session_id, api_code, dockerfile, requirements):
    api_dir = Path(f"sessions/{session_id}/api")
    api_dir.mkdir(parents=True, exist_ok=True)

    (api_dir / "app.py").write_text(api_code)
    (api_dir / "Dockerfile").write_text(dockerfile)
    (api_dir / "requirements.txt").write_text(requirements)

    # Copy model files into api directory for Docker portability
    import shutil
    models_src = Path(f"sessions/{session_id}/models")
    models_dst = api_dir / "models"
    if models_src.exists():
        shutil.copytree(models_src, models_dst, dirs_exist_ok=True)

    return api_dir


def start_api(api_dir, port=8000):
    """Start the FastAPI server as a background process."""
    process = subprocess.Popen(
        ["uvicorn", "app:app", "--host", "0.0.0.0",
         "--port", str(port), "--log-level", "error"],
        cwd=str(api_dir),
        stdout=subprocess.PIPE,
        stderr=subprocess.PIPE
    )
    # Wait for startup
    time.sleep(3)
    return process, port


def find_available_port(start=8000, end=8010):
    import socket
    for port in range(start, end):
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            if s.connect_ex(("localhost", port)) != 0:
                return port
    raise RuntimeError("No available ports in range 8000–8010.")
```

## Health Check and Test Prediction

```python
def run_health_check(port, retries=5, delay=2):
    url = f"http://localhost:{port}/health"
    for attempt in range(retries):
        try:
            response = requests.get(url, timeout=5)
            if response.status_code == 200:
                return True, response.json()
        except Exception:
            pass
        time.sleep(delay)
    return False, {"error": "API did not respond after multiple attempts."}


def run_test_prediction(port, sample_row, feature_columns):
    """Send a sample row to the API and verify it responds correctly."""
    url     = f"http://localhost:{port}/predict"
    payload = {"data": {col: val for col, val in
                        zip(feature_columns, sample_row)}}
    try:
        response = requests.post(url, json=payload, timeout=10)
        if response.status_code == 200:
            return True, response.json()
        else:
            return False, response.json()
    except Exception as e:
        return False, {"error": str(e)}
```

## Running the Full Deployment Pipeline

```python
def run_deployment(model_path, scaler_path, feature_columns,
                   target_col, task_type, X_test,
                   session_id):

    # 1. Generate API code
    api_code    = generate_api_code(model_path, scaler_path,
                                     feature_columns, target_col,
                                     task_type, session_id)
    dockerfile  = generate_dockerfile(session_id)
    requirements = generate_requirements()

    # 2. Write files
    api_dir = write_api_files(session_id, api_code,
                               dockerfile, requirements)

    # 3. Start API
    port    = find_available_port()
    process, port = start_api(api_dir, port)

    # 4. Health check
    healthy, health_response = run_health_check(port)
    if not healthy:
        process.terminate()
        return {
            "status": "failed",
            "error": "The API did not start correctly.",
            "plain_english": (
                "The API did not start correctly. This is usually caused by "
                "a missing dependency. Check that all required packages are "
                "installed and try again."
            )
        }

    # 5. Test prediction
    sample_row = X_test.iloc[0].tolist()
    test_passed, test_response = run_test_prediction(
        port, sample_row, feature_columns
    )

    return {
        "status":        "success" if test_passed else "degraded",
        "api_url":       f"http://localhost:{port}",
        "health_url":    f"http://localhost:{port}/health",
        "predict_url":   f"http://localhost:{port}/predict",
        "features_url":  f"http://localhost:{port}/features",
        "docs_url":      f"http://localhost:{port}/docs",
        "api_dir":       str(api_dir),
        "port":          port,
        "test_passed":   test_passed,
        "test_response": test_response,
        "process":       process
    }
```
